# Changelog

The full release history (with commit-level details) lives on GitHub Releases:

- [Wybthon releases on GitHub](https://github.com/wybthon/wybthon/releases)

Each release page lists the highlights, breaking changes, and a diff
link to the previous tag. Subscribe via GitHub's "Watch → Releases
only" to be notified of new versions.

## How releases work

- We follow [Semantic Versioning](https://semver.org/): `MAJOR.MINOR.PATCH`.
- `0.x` releases may include breaking changes in minor versions while the API stabilizes.
- Each release is published to PyPI and tagged in git.
